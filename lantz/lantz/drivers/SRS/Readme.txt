DG645 Readme

1. You could press '3' to switch back to front panel access during remote control

2. If error occurs, use the Clear_Status command

3. To recall the original settings, press 'RCL'-->'0'-->'Enter' on the front panel

4. 'Trigger_Level' means External trigger threshold

5. Switch between different modes by commands like: inst.Trigger_Source='External rising edges'

6. You could set each pulse edge through 'Delay Channel'     0--T0   1--T1   2--A   3--B   4--C   5--D   6--E  7--F  8--G  9--H
